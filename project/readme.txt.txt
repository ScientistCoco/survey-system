- The tests are run by running the command line: python -m pytest 
	- The test file should be contained in the directory of the application files
- The tests contained in tests.zip include: 
	- Test for creating mandatory questions
	- Test for adding questions to a course survey
	- Testing the enrolment of students into a course
	- Testing the change of a course survey status as an admin role
	- Testing the change of a course survey status as a teacher role
